About This Project

EV Analytics is a Business Intelligence platform developed for analyzing Electric Vehicle data using modern technologies.

MySQL Database

Stores Electric Vehicle datasets, charging information, battery specifications, and vehicle performance records.

Tableau Dashboard

Interactive dashboards and stories provide meaningful insights into EV charging, battery performance, and efficiency.

Flask Application

Integrates Tableau dashboards into a responsive web application with an easy-to-use interface.
