# EV Analytics Flask

This is a simple Flask web application for exploring EV analytics content.

## Run locally

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## Deploy to GitHub

1. Create a new repository on GitHub.
2. Connect this local repository with:
   ```bash
   git remote add origin <your-github-repo-url>
   git branch -M main
   git push -u origin main
   ```
